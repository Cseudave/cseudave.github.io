Limpieza de base de datos FIFA21

Este proyecto consistió en la limpieza y remodelación del dataset para poder trabajar con él. Se revisarón sus más de 70 columnas una por una.

  1. Se creo una función que mostró datos importantes y ejemplos de cada columna, preguntando cambios sugeridos
  2. Se siguió el documento creado y se abordarón las 77 columnas usando pandas y regex
  3. Se compusieron los nombres obteniendo información de name, long_name y playerUrl, cambiando los nombres abreviados a su versión completa
  4. Se eliminó el único valor duplicado
  5. Se transformó la columna "Team & Contract" en cuatro columnas más utiles
  6. Se pasarón las columnas a unidades metricas internacionales
  7. Se convirtierón los valores monetarios en cantidad de euros, haciendo la columna numerica.
  8. Se revisó las columnas de estadísticas con una gráfica de caja y un histograma para observar si se trataban de distribuciones razonables
